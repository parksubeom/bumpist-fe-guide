# bumpist-fe-guide

프론트엔드(Vue·React) 작업의 공통 규칙·스킬·규약을 모아 둔 개인 가이드 허브다.
다른 프로젝트에서 git submodule로 참조해 규칙과 Claude 스킬을 한 번에 얹는다.

가이드 본문은 [claude/README.md](./claude/README.md)에 있다. 적용법은 [claude/install.md](./claude/install.md).

```sh
git submodule add git@github.com:parksubeom/bumpist-fe-guide.git .fe-guide
sh .fe-guide/claude/apply-to-project.sh vue   # 또는 react
```
