# 쉬운 설명 — 비개발자·초보자용

> 개발이 처음이어도 **그대로 따라만 하면** 되게 쓴 문서예요. 순서대로 하세요.
> 이미 개발을 좀 아신다면 [README](./README.md)로 가도 됩니다.

## 이게 뭐예요?

**AI(Claude)와 함께 웹 화면을 만들 때, 매번 같은 방식으로 잘 만들도록 도와주는 "설명서 + 자동 도우미 모음"** 이에요.

비유하자면 — 주문할 때마다 요리 맛이 달라지면 곤란하죠. 그래서 **레시피(규칙)** 를 정해두고 **요리사(AI)** 가
그대로 만들게 하는 거예요. 규칙을 외울 필요 없이, **하고 싶은 걸 말로 시키면** AI가 알아서 규칙대로 해줘요.

---

## 0단계 — 준비물 (처음이면 여기부터)

컴퓨터에 딱 3개만 준비하면 돼요. 이미 있으면 건너뛰세요.

### (1) 터미널 열기

**터미널**은 컴퓨터에 글자로 명령을 시키는 창이에요. 앞으로 명령을 여기에 붙여넣습니다.

- **맥**: `⌘(커맨드) + 스페이스` → `터미널` 입력 → Enter
- **윈도우**: 아래 (2)에서 git을 깔면 **"Git Bash"** 가 같이 설치돼요. 시작 메뉴에서 `Git Bash`를 열어
  거기서 하세요. (윈도우는 이 문서의 명령을 **Git Bash 창**에서 실행)

### (2) git 과 Node 설치 (딱 두 개)

터미널에 아래를 쳐서 **숫자(버전)** 가 나오면 이미 깔린 거예요:

```
git --version
node --version
```

- `git`이 없다고 나오면 → [git-scm.com](https://git-scm.com) 에서 설치 (윈도우는 설치 시 "Git Bash"도 같이 깔림).
- `node`가 없다고 나오면 → [nodejs.org](https://nodejs.org) 에서 **LTS** 버튼으로 설치.

### (3) Claude Code 설치

Node를 깔았으면 터미널에:

```
npm install -g @anthropic-ai/claude-code
```

(맥에서 권한 오류가 나면 앞에 `sudo `를 붙여 다시 실행) 설치되면 `claude` 라고 쳤을 때 Claude가 열려요.
처음엔 로그인하라고 나오는데 안내대로 따라가면 됩니다.

---

## 1단계 — 내 프로젝트 준비

내 작업이 담길 **폴더**가 필요해요.

- **이미 만들던 프로젝트가 있으면** 그 폴더로 이동:
  ```
  cd 폴더경로
  ```
- **아직 없으면** 새로 만들기:
  ```
  mkdir 내프로젝트
  cd 내프로젝트
  git init
  ```

## 2단계 — 가이드 복사 (한 줄씩 붙여넣기)

아래를 **한 줄씩** 붙여넣고 Enter (윈도우는 Git Bash에서). Vue 기준이고, React면 `vue`를 `react`로 바꾸세요.

```
git clone --depth 1 https://github.com/parksubeom/bumpist-fe-guide.git .bumpist
```

→ 가이드를 잠깐 내려받아요.

```
sh .bumpist/apply-to-project.sh vue .bumpist
```

→ 규칙·스킬을 내 프로젝트로 복사해요. 이때 화면에 **`@`로 시작하는 줄들**이 주르륵 나와요 — **그 줄들을
잠깐 복사(선택 후 Ctrl/⌘+C)** 해 두세요.

```
rm -rf .bumpist
```

→ 다 썼으니 임시 폴더를 지워요.

## 3단계 — CLAUDE.md에 붙여넣기

**CLAUDE.md** 는 Claude가 작업 전에 읽는 "지침 쪽지"예요.

1. 프로젝트 폴더 안에 `CLAUDE.md` 라는 이름의 파일을 새로 만든다 (메모장·VS Code 등 아무 텍스트 편집기로,
   저장할 때 이름을 정확히 `CLAUDE.md`로).
2. 2단계에서 복사해 둔 `@...` 줄들을 그 안에 붙여넣고 저장한다.

## 4단계 — 이제 시키기만 하면 끝!

프로젝트 폴더에서 `claude` 를 실행하고, 하고 싶은 걸 **말로** 하세요:

- "뭐부터 하면 돼?"
- "버튼 하나 만들어줘"
- "회원 목록 화면 만들어줘"

규칙은 Claude가 알아서 지켜요. 끝이에요.
