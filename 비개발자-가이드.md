# 쉬운 설명 — 비개발자·초보자용

> 개발이 처음이거나 코드를 잘 몰라도 이해할 수 있게 쓴 문서예요.
> 개발자라면 [README](./README.md)로 가세요.

## 이게 뭐예요?

한마디로 **"AI(Claude)와 함께 웹 화면을 만들 때, 매번 같은 방식으로 잘 만들도록 도와주는 설명서 +
자동 도우미 모음"** 이에요.

비유하자면 — 주문할 때마다 요리 맛이 달라지면 곤란하죠. 그래서 **레시피(규칙)** 를 정해두고,
**요리사(AI)** 가 그 레시피대로 만들게 하는 거예요. 이 저장소가 그 레시피와 요리사 사용법을 모아둔 곳이에요.

- **규칙(rules)** = 레시피. "우리는 이렇게 만든다"는 약속.
- **스킬(skills)** = 요리사에게 시키는 자동 기능. "메뉴 하나 추가해줘" 하면 레시피대로 뚝딱 만들어 줘요.

## 왜 필요해요?

사람마다, 그때그때 다른 방식으로 만들면 나중에 서로 못 알아보고 엉켜요. 규칙을 한 번 정해두면:

- 누가 만들든 결과물이 비슷해져요.
- 처음 온 사람도 금방 따라 할 수 있어요.
- AI에게 "알아서 규칙대로 해줘" 하고 맡길 수 있어요.

## 뭘 할 수 있어요?

Claude(AI)에게 **말로 시키면** 규칙대로 해줘요. 예를 들면:

- "버튼 하나 만들어줘" → 정해진 방식대로 버튼 + 점검(테스트)까지
- "회원 목록 화면 만들어줘" → 화면 + 데이터 연결
- "이거 저장(커밋)해줘" → 규칙 검사 후 정리해서 저장

규칙을 외울 필요 없어요. **하고 싶은 걸 그냥 말하면** AI가 알맞은 규칙과 기능을 찾아 써요.
