# 쉬운 설명 — 비개발자·초보자용

> 개발이 처음이어도 **그대로 따라만 하면** 되게 쓴 문서예요. 순서대로 하세요.
> 이미 개발을 좀 아신다면 [README](./README.md)로 가도 됩니다.

## 이게 뭐예요?

**AI(Claude)와 함께 웹 화면을 만들 때, 매번 같은 방식으로 잘 만들도록 도와주는 "설명서 + 자동 도우미 모음"** 이에요.

비유하자면 — 주문할 때마다 요리 맛이 달라지면 곤란하죠. 그래서 **레시피(규칙)** 를 정해두고 **요리사(AI)** 가
그대로 만들게 하는 거예요. 규칙을 외울 필요 없이, **하고 싶은 걸 말로 시키면** AI가 알아서 규칙대로 해줘요.

---

## 0단계 — 준비물 (처음이면 여기부터)

컴퓨터에 딱 3개만 준비하면 돼요. 이미 있으면 건너뛰세요.

### (1) 터미널 열기

**터미널**은 컴퓨터에 글자로 명령을 시키는 창이에요. 앞으로 명령을 여기에 붙여넣습니다.

- **맥**: `⌘(커맨드) + 스페이스` → `터미널` 입력 → Enter
- **윈도우**: 시작 메뉴에서 **`PowerShell`** 을 열면 돼요. (또는 아래 (2)에서 git 설치 시 같이 깔리는
  `Git Bash` — 둘 중 아무거나. 이 문서 명령은 둘 다에서 동작)

![스크린샷: 맥 스포트라이트에서 "터미널" 검색 / 윈도우 시작 메뉴에서 "Git Bash" 여는 화면](./assets/screenshots/placeholder.svg)

### (2) git 과 Node 설치 (딱 두 개)

터미널에 아래를 쳐서 **숫자(버전)** 가 나오면 이미 깔린 거예요:

```
git --version
node --version
```

- `git`이 없다고 나오면 → [git-scm.com](https://git-scm.com) 에서 설치 (윈도우는 설치 시 "Git Bash"도 같이 깔림).
- `node`가 없다고 나오면 → [nodejs.org](https://nodejs.org) 에서 **LTS** 버튼으로 설치.

![스크린샷: 터미널에 git --version, node --version 을 쳐서 버전 숫자가 나오는 화면](./assets/screenshots/placeholder.svg)

### (3) Claude Code 설치

Node를 깔았으면 터미널에:

```
npm install -g @anthropic-ai/claude-code
```

(맥에서 권한 오류가 나면 앞에 `sudo `를 붙여 다시 실행) 설치되면 `claude` 라고 쳤을 때 Claude가 열려요.
처음엔 로그인하라고 나오는데 안내대로 따라가면 됩니다.

![스크린샷: claude 실행 후 로그인/시작 화면](./assets/screenshots/placeholder.svg)

## 1단계 — 내 프로젝트 준비

내 작업이 담길 **폴더**가 필요해요.

- **이미 만들던 프로젝트가 있으면** 그 폴더로 이동:
  ```
  cd 폴더경로
  ```
- **아직 없으면** 새로 만들기:
  ```
  mkdir 내프로젝트
  cd 내프로젝트
  git init
  ```

## 2단계 — 가이드 설치 (명령 한 줄)

프로젝트 폴더 안에서 아래 **한 줄**을 붙여넣고 Enter (Vue면 `vue`, React면 `react`, Next.js면 `next`):

```
npx bumpist-code@latest init vue
```

→ 규칙·스킬이 `.claude/` 로 복사되고, **`CLAUDE.md`(Claude가 작업 전에 읽는 "지침 쪽지")가 자동으로
만들어져요.** 처음 실행 시 설치할지 물으면 `y`. (이미 `CLAUDE.md`가 있으면 추가할 줄만 알려줘요 — 그대로
복사해 붙여넣으면 됩니다.)

![스크린샷: npx bumpist-code init vue 실행 후 복사 완료 메시지 화면](./assets/screenshots/placeholder.svg)

## 3단계 — 이제 시키기만 하면 끝!

프로젝트 폴더에서 `claude` 를 실행하고, 하고 싶은 걸 **말로** 하세요:

- "뭐부터 하면 돼?"
- "버튼 하나 만들어줘"
- "회원 목록 화면 만들어줘"

규칙은 Claude가 알아서 지켜요. 끝이에요.

![스크린샷: 프로젝트 폴더에서 claude 실행 후 "버튼 하나 만들어줘"라고 말하는 화면](./assets/screenshots/placeholder.svg)

## 겁먹지 마세요 / 막히면

- 규칙을 다 읽거나 외울 필요 **없어요**. 필요할 때 Claude가 알아서 찾아 써요.
- 명령을 쳤는데 **빨간 글씨(에러)** 가 나오면, 그 글씨를 **그대로 복사해서 Claude에게 붙여넣고 "이거 왜 이래?"**
  하고 물어보세요. 원인과 해결을 알려줘요.
- 뭘 먼저 할지 모르겠으면 Claude에게 **"뭐부터 하면 돼?"** 라고 물어보세요.
